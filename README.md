## 🍗 Sobre Cali Pollo

Cali Pollo es un restaurante especializado en platillos a base de pollo con sabores tradicionales de la cocina colombiana. Este proyecto busca desarrollar una plataforma digital para mejorar la experiencia de sus clientes y optimizar sus procesos internos.
