Estándares de Nomenclatura para Variables y Constantes

Este documento establece las convenciones de nomenclatura para variables y constantes en el proyecto Cali Pollo. Estas pautas ayudarán a mantener un código consistente, legible y mantenible para todos los desarrolladores del equipo.

Variables y Constantes en JavaScript

Variables

Las variables en JavaScript usarán la convención camelCase:

// Variables simples
let menuPrice = 15000;
let itemName = 'Pollo Asado';
let isAvailable = true;

// Variables de objetos complejos
let activeOrder = {
    items: [],
    customer: {},
    total: 0
};

// Arrays
let popularDishes = ['Pollo Asado', 'Pollo Broaster', 'Alitas BBQ'];

Prefijos y sufijos recomendados:

  • is/has/can - Para variables booleanas
  • min/max - Para límites
  • num/count - Para cantidades numéricas
  • temp - Para variables temporales
// Ejemplos específicos para Cali Pollo
let isSpicy = true;
let hasDelivery = true;
let canApplyDiscount = false;
let minOrderValue = 20000;
let maxDeliveryDistance = 10;
let numOrderItems = 5;
let tempTotalPrice = 45000;

Constantes

Las constantes en JavaScript seguirán dos convenciones:

1. Constantes primitivas (valores fijos que no cambiarán):

// Usar UPPER_SNAKE_CASE para constantes "verdaderas" que representan valores fijos
const TAX_RATE = 0.19;
const DELIVERY_FEE = 5000;
const MIN_PURCHASE_AMOUNT = 15000;
const MAX_ITEMS_PER_ORDER = 20;

// Constantes específicas de Cali Pollo
const RESTAURANT_NAME = 'Cali Pollo';
const RESTAURANT_PHONE = '601-555-1234';
const RESTAURANT_ADDRESS = 'Calle 15 #25-30, Cali';
const OPENING_HOUR = 11;
const CLOSING_HOUR = 22;

2. Referencias constantes (declaradas con const pero su contenido puede cambiar):

// Usar camelCase para referencias const a objetos y arrays
const menuConfig = {
    showPrices: true,
    defaultCategory: 'platos-principales',
    sortOrder: 'price'
};

const deliveryZones = [
    { name: 'Zona Norte', fee: 5000 },
    { name: 'Zona Sur', fee: 6000 },
    { name: 'Zona Este', fee: 7000 },
    { name: 'Zona Oeste', fee: 8000 }
];

DOM y jQuery Variables

Para elementos DOM, usar el prefijo $ para identificarlos claramente:

// Variables DOM con prefijo $
const $menuContainer = document.getElementById('menu-container');
const $orderForm = document.querySelector('.order-form');
const $priceElements = document.querySelectorAll('.menu-item__price');

// Con jQuery (si se usa)
const $menuItems = $('.menu-item');
const $submitButton = $('#submit-order');

Variables y Constantes en PHP

Variables

Las variables en PHP utilizarán camelCase:

<?php
// Variables simples
$menuItem = 'Pollo Asado';
$itemPrice = 18000;
$isSpecialOffer = true;

// Arrays
$menuCategories = ['Platos Principales', 'Entradas', 'Postres', 'Bebidas'];

// Variables dentro de clases
class Order
{
    private $orderId;
    private $customerName;
    private $orderItems = [];
    private $totalAmount = 0;

    public function addItem($itemName, $itemPrice)
    {
        $newItem = [
            'name' => $itemName,
            'price' => $itemPrice
        ];

        $this->orderItems[] = $newItem;
        $this->updateTotal();
    }
}

Constantes

Las constantes en PHP se definirán en UPPER_SNAKE_CASE:

<?php
// Constantes globales
define('TAX_RATE', 0.19);
define('DELIVERY_FEE_BASE', 5000);
define('RESTAURANT_NAME', 'Cali Pollo');

// Constantes de clase (PHP 7.1+)
class PaymentProcessor
{
    const PAYMENT_METHOD_CASH = 'cash';
    const PAYMENT_METHOD_CREDIT = 'credit';
    const PAYMENT_METHOD_DEBIT = 'debit';
    const PAYMENT_METHOD_ONLINE = 'online';

    const PAYMENT_STATUS_PENDING = 'pending';
    const PAYMENT_STATUS_COMPLETED = 'completed';
    const PAYMENT_STATUS_FAILED = 'failed';

    // ...
}

// Constantes en Enum (PHP 8.1+)
enum OrderStatus: string
{
    case PENDING = 'pending';
    case PROCESSING = 'processing';
    case COMPLETED = 'completed';
    case DELIVERED = 'delivered';
    case CANCELED = 'canceled';
}

Variables en CSS

Variables CSS (Custom Properties)

Para las variables CSS (propiedades personalizadas), utilizaremos la siguiente convención:

:root {
    /* Colores de la marca Cali Pollo */
    --color-primary: #e8a628;     /* Amarillo/naranja pollo */
    --color-secondary: #dd5928;   /* Rojo anaranjado picante */
    --color-accent: #8c0303;      /* Rojo oscuro salsa */
    --color-dark: #333333;        /* Oscuro para texto */
    --color-light: #f4f4f4;       /* Claro para fondos */

    /* Tipografía */
    --font-main: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    --font-headings: 'Georgia', serif;
    --font-size-base: 16px;
    --font-size-h1: 2.5rem;
    --font-size-h2: 2rem;
    --font-size-h3: 1.5rem;

    /* Espaciado */
    --spacing-xs: 0.25rem;
    --spacing-sm: 0.5rem;
    --spacing-md: 1rem;
    --spacing-lg: 2rem;
    --spacing-xl: 3rem;

    /* Bordes y sombras */
    --border-radius-sm: 3px;
    --border-radius-md: 5px;
    --border-radius-lg: 10px;
    --shadow-card: 0 2px 5px rgba(0, 0, 0, 0.1);
}

/* Uso de las variables */
.menu-item {
    background-color: white;
    border-radius: var(--border-radius-md);
    padding: var(--spacing-md);
    box-shadow: var(--shadow-card);
}

.menu-item__title {
    color: var(--color-accent);
    font-family: var(--font-headings);
    font-size: var(--font-size-h3);
}

.menu-item--featured {
    border-left: 5px solid var(--color-primary);
}

Convención de nomenclatura:

  • Nombres en minúsculas
  • Palabras separadas por guiones
  • Agrupación por categoría (color, font, spacing, etc.)
  • Para variables relacionadas, usar sufijos como xs, sm, md, lg, xl

Ejemplos completos para el proyecto

Aquí hay algunos ejemplos de código con variables nombradas según nuestras convenciones:

// JavaScript - Sistema de pedido
const menuItems = [
  { id: 1, name: 'Pollo Asado', price: 18000, isSpicy: false, category: 'platos-principales' },
  { id: 2, name: 'Alitas BBQ', price: 15000, isSpicy: true, category: 'entradas' },
  { id: 3, name: 'Pollo Broaster', price: 20000, isSpicy: false, category: 'platos-principales' }
];

const DELIVERY_MIN_PURCHASE = 15000;
const TAX_RATE = 0.19;

let orderItems = [];
let orderSubtotal = 0;
let orderTaxAmount = 0;
let orderTotal = 0;
let customerName = '';
let customerPhone = '';
let customerAddress = '';
let isDeliveryOrder = false;

function calculateOrderTotal() {
  orderSubtotal = orderItems.reduce((sum, item) => sum + item.price, 0);
  orderTaxAmount = orderSubtotal * TAX_RATE;
  orderTotal = orderSubtotal + orderTaxAmount;

  if (isDeliveryOrder && orderSubtotal < DELIVERY_MIN_PURCHASE) {
    throw new Error(`El pedido mínimo para domicilio es $${DELIVERY_MIN_PURCHASE}`);
  }

  return orderTotal;
}
                    
// PHP - Clase para Item del Menú
<?php
class MenuItem
{
    private $menuItemId;
    private $menuItemName;
    private $menuItemPrice;
    private $menuItemDescription;
    private $isAvailable;
    private $isSpicy;
    private $menuItemImageUrl;
    private $menuItemCategory;
    private $menuItemCalories;

    // Constantes de categorías
    const CATEGORY_MAIN = 'platos-principales';
    const CATEGORY_STARTER = 'entradas';
    const CATEGORY_SIDE = 'acompanantes';
    const CATEGORY_DESSERT = 'postres';
    const CATEGORY_DRINK = 'bebidas';

    public function __construct($menuItemId, $menuItemName, $menuItemPrice)
    {
        $this->menuItemId = $menuItemId;
        $this->menuItemName = $menuItemName;
        $this->menuItemPrice = $menuItemPrice;
        $this->isAvailable = true;
    }

    public function getMenuItemPrice()
    {
        return $this->menuItemPrice;
    }

    public function isMenuItemAvailable()
    {
        return $this->isAvailable;
    }

    public function setMenuItemCategory($categoryName)
    {
        $validCategories = [
            self::CATEGORY_MAIN,
            self::CATEGORY_STARTER,
            self::CATEGORY_SIDE,
            self::CATEGORY_DESSERT,
            self::CATEGORY_DRINK
        ];

        if (in_array($categoryName, $validCategories)) {
            $this->menuItemCategory = $categoryName;
            return true;
        }

        return false;
    }
}
